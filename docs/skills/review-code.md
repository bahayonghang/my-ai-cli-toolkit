# Review Code

Multi-dimensional code review with structured reports and actionable recommendations.

## Overview

Review Code is a comprehensive code review skill that analyzes code across six key dimensions: correctness, readability, performance, security, testing, and architecture. It generates structured review reports with specific, actionable recommendations.

## Features

- 🔍 **6-Dimension Analysis** - Comprehensive review across multiple aspects
- 📊 **Structured Reports** - Clear, organized findings with severity levels
- 🎯 **Actionable Recommendations** - Specific suggestions with code examples
- 🔒 **Security Focus** - Identifies vulnerabilities and security issues
- ⚡ **Performance Insights** - Spots optimization opportunities
- 🏗️ **Architecture Review** - Evaluates design patterns and structure

## Review Dimensions

### 1. Correctness
- Logic errors
- Edge cases
- Error handling
- Type safety

### 2. Readability
- Code clarity
- Naming conventions
- Comments and documentation
- Code organization

### 3. Performance
- Algorithm efficiency
- Resource usage
- Bottlenecks
- Optimization opportunities

### 4. Security
- Vulnerabilities
- Input validation
- Authentication/authorization
- Data protection

### 5. Testing
- Test coverage
- Test quality
- Edge case testing
- Test maintainability

### 6. Architecture
- Design patterns
- SOLID principles
- Modularity
- Scalability

## Review Process

1. **Specification Study** - Understands requirements and context
2. **Multi-Dimensional Analysis** - Evaluates across all six dimensions
3. **Report Generation** - Creates structured findings document
4. **Recommendations** - Provides specific improvement suggestions

## Best Practices

- Provide context about the code's purpose
- Include related files for better understanding
- Specify areas of concern if any
- Review the report and ask follow-up questions
