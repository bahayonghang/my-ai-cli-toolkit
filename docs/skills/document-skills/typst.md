# Typst

General Typst authoring skill for generating and editing idiomatic `.typ` documents.

## When to use it

- create new Typst documents
- update existing `.typ` sources
- verify syntax against bundled Typst docs
- compile a Typst document when tool access allows

## Workflow

1. trust the local Typst documentation bundled with the skill
2. read the guides, tutorial, or reference pages relevant to the request
3. generate or edit the `.typ` file
4. compile with `typst compile` when available

## Bundled documentation

- `docs/guides/`
- `docs/tutorial/`
- `docs/reference/`

## Notes

- This skill is intentionally broad, while `touying` is the presentation-specific companion.
