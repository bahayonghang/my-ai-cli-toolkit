# AlphaXiv Paper Lookup

Look up any arxiv paper on alphaxiv.org to get a structured AI-generated overview.

## When to use it

- user shares an arxiv URL or paper ID
- quick explanation or summary of an arxiv paper
- fetching AI-generated overviews without reading the full PDF

## Workflow

1. extract the paper ID from the provided URL or input
2. resolve the paper via the alphaxiv API
3. fetch the AI-generated overview
4. optionally fetch the full text for deeper analysis

## Main assets

- alphaxiv.org API integration

## Notes

- No authentication required.
- Supports multiple languages including en, zh, ja, and others.
