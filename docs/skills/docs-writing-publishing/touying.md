# Touying

Typst slide authoring skill for Touying-based presentations, themes, layout, and animation.

## When to use it

- edit `.typ` presentation files
- structure a Touying deck
- configure themes and global settings
- add slide transitions or staged reveals

## What the skill emphasizes

- theme import and `#show`-based configuration
- multi-file deck organization with shared globals
- Touying-specific layout and animation rules
- references to the bundled docs under `docs/` and `references/`

## Key rules

- use Touying config helpers instead of raw `set page`
- keep shared deck config centralized
- provide a valid `.typ` target path before editing
