# BidWriter

Bid-writing workflow for engineering consulting, architecture, municipal engineering, and related tender documents.

## When to use it

- technical or commercial bid drafting
- tender document analysis
- scoring-rule extraction and response planning
- proposal writing for engineering-heavy tenders

## Core workflow

1. collect project basics, tender file, bid type, company profile, and formatting constraints
2. analyze requirements and scoring items
3. map the tender structure into a working document tree
4. draft section-by-section responses with strong alignment to the original tender numbering
5. perform compliance, scoring, and consistency review before delivery

## Main supporting assets

- `references/SCORING_GUIDE.md`
- `references/CHAPTER_TEMPLATES.md`
- `references/TERMINOLOGY.md`
- `references/STANDARDS.md`

## Key rule

Do not invent credentials, past projects, or certificates. Mark missing company data explicitly as pending.
