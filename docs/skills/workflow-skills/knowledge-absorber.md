# Knowledge Absorber

Deep analysis of links, documents, or code to generate comprehensive teaching notes from zero to mastery.

## When to use it

- deep learning from URLs, documents, or code repositories
- generating structured teaching materials
- building a personal knowledge base from external sources

## Workflow

1. ingest content via the built-in script (URLs, files, or code)
2. load the teaching persona
3. generate dual-mode output: Markdown for reading and HTML for interactive viewing

## Main assets

- three-level content loading mechanism
- dual output format (Markdown + HTML)

## Notes

- Outputs are saved to a `knowledge_{date}_{title}/` directory.
