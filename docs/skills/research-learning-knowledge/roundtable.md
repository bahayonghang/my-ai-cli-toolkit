# Roundtable

Multi-perspective discussion skill for examining one topic through a moderator and a small set of real figures.

## When to use it

- user asks for a roundtable, structured debate, or multi-voice discussion
- user wants thinkers or public figures to clash around one question
- user wants a disciplined exploration instead of a plain summary

## Core workflow

1. narrow the request to one central topic
2. choose 3-4 real figures whose positions create meaningful tension
3. open with moderator framing and a shared definition question
4. run one focused round rather than an endless simulated conversation
5. summarize the deepest disagreement and draw a small ASCII structure map
6. let the user decide whether to continue, stop, go deeper, or introduce a new figure
7. optionally export the discussion as an Org-mode note

## Output contract

- the first response should include moderator framing, participant list, the first round, a moderator summary, and a control menu
- stance-based paraphrase is preferred over fabricated quotes
- exported notes must stay in pure Org-mode when file writing is available

## Main supporting assets

- `content/skills/research-learning-knowledge/roundtable/SKILL.md`
- `content/skills/research-learning-knowledge/roundtable/references/original-prompt.org`

## Key constraints

- use real figures conservatively and only when their relevance is defensible
- do not invent exact quotes, citations, or false historical details
- ASCII only for structural diagrams
- keep each round focused on one deep tension

## Notes

- This skill is for guided intellectual collision, not generic roleplay.
- The moderator should stop after the first round unless the user asks to continue.
