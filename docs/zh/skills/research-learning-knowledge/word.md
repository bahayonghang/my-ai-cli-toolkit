# word: 英文单词深解

把一个英文单词拆到语义骨架、原始画面、现代用法和易混点，而不是只给翻译。适合真正想掌握一个词的人。

## 适用场景

- 用户要深度解释某个英文单词
- 需要词源、意象、现代用法和一句顿悟
- 想要 Markdown 格式的单词解析结果

## 工作流程

1. 标准化单词并确认标题行。
2. 写出 surface anchor、原始画面和核心意象。
3. 展开深层解释，再补 usage map。
4. 对比易混词或常见误用。
5. 最后用一句中英双语的句子收束。

## 主要资源

- 单词的词源与语义演化
- 音标、翻译和核心意象
- 固定的 Markdown 输出结构

## 关键约束

- 目标是 mastery，不是直译
- 输出要有洞见，不能停留在字典释义
- 必须覆盖用法地图和易混点
- 只处理一个单词，不处理短语或多词输入
