# word-flow: 词卡流

一条命令完成“解词 + 做信息图”：先把单词讲透，再把结果铸成 PNG 词卡，支持多个单词并行。

## 适用场景

- 用户给出一组明确的英文单词
- 既想要单词解析，又想要信息图卡片
- 想把英文词学习结果直接转成可分享图片

## 工作流程

1. 只收集明确指定的目标词，不把整句英文里的所有 token 都算进去。
2. 对每个单词先执行 `word`，拿到深解文本。
3. 再以该解析结果为输入，调用 `card -i` 生成信息图 PNG。
4. 多个单词可以并行处理，最后汇总每个单词的图片路径。

## 主要资源

- `word`：单词深解主技能
- `card -i`：信息图模具
- `~/Downloads/`：词卡输出目录

## 关键约束

- 必须先解词，再做卡片
- 卡片内容来源于解析结果，不直接用字典释义
- 单词之间可并行，但单词内部流程必须串行
- 依赖的是 `word` 和 `card`，不是旧的 `ljg-*` 名称
- 若某个词的卡片失败，要报告失败原因，不要伪造路径
