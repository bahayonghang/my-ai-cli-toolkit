# skill-evolution-manager

根据用户反馈持续迭代优化技能。

## 概述

Skill Evolution Manager 是一个专门用于在对话结束时收集用户反馈并迭代优化现有技能的工具。它分析对话内容，提取改进建议，并自动更新技能定义。

## 功能特性

- 📊 **反馈收集** - 在对话结束时自动收集用户反馈
- 🔍 **内容分析** - 分析对话中的使用模式和问题
- ✏️ **智能优化** - 基于反馈生成改进建议
- 📈 **版本追踪** - 记录技能演进历史
- 🔄 **自动更新** - 将改进应用到技能定义

## 使用方法

```
/skill-evolution-manager
```

```
根据这次对话优化技能
```

## 工作流程

1. **会话分析**
   - 回顾对话历史
   - 识别使用的技能
   - 检测问题和障碍

2. **反馈提取**
   - 用户明确反馈
   - 隐式使用模式
   - 失败或重试情况

3. **改进生成**
   - 识别改进机会
   - 生成具体修改建议
   - 评估影响范围

4. **应用更新**
   - 更新 SKILL.md
   - 添加新示例
   - 优化指导说明

## 演进类型

| 类型 | 描述 | 示例 |
|------|------|------|
| 文档改进 | 改善描述清晰度 | 添加更多示例 |
| 功能扩展 | 新增使用场景 | 支持新格式 |
| 问题修复 | 修正错误指导 | 更正命令语法 |
| 优化调整 | 改善用户体验 | 简化工作流程 |

## 版本管理

每次演进都会记录：
- 变更内容
- 触发原因
- 影响评估
- 回滚信息

## 最佳实践

1. **定期演进** - 根据使用反馈持续优化
2. **小步迭代** - 每次只做小范围改进
3. **保留历史** - 记录所有变更以便回溯
4. **测试验证** - 确保改进不破坏现有功能
