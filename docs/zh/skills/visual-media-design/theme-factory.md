# 主题工厂 (Theme Factory)

为幻灯片、报告、文档、HTML 页面等产物统一应用视觉主题的技能。

## 适用场景

- 用户希望为幻灯片、报告、文档或其他生成产物套用统一主题
- 已有产物需要应用一个精心设计的视觉风格
- 现成主题都不合适，需要即时生成一个新主题

## 工作流程

1. 先展示内置的 `theme-showcase.pdf`，让用户查看可用主题
2. 要求用户明确选择主题
3. 读取 `themes/` 中对应的主题定义
4. 将主题中的颜色与字体统一应用到目标产物
5. 如果预设主题都不合适，则生成自定义主题并在应用前确认

## 内置主题

该技能提供十套预设主题，例如 Ocean Depths、Sunset Boulevard、Modern Minimalist、Tech Innovation 和 Midnight Galaxy。

## 主要资源

- `theme-showcase.pdf`：主题总览文件
- `themes/`：各主题的详细定义

## 说明

- 该技能不限定产物类型，可用于幻灯片、文档、报告或 HTML 页面。
- 应用预设主题前，需要用户显式确认所选主题。
- 如果现有主题都不合适，技能可以现场生成并应用新主题。
