# Typst

通用的 Typst 编写技能，用于生成和编辑符合惯用法的 `.typ` 文档。

## 适用场景

- 创建新的 Typst 文档
- 修改已有 `.typ` 文件
- 先对照本地 Typst 文档再生成代码
- 在允许的情况下编译 Typst 文档

## 工作流程

1. 优先相信技能自带的本地 Typst 文档
2. 读取与任务相关的 guide、tutorial、reference 页面
3. 生成或编辑 `.typ`
4. 在可用时执行 `typst compile`

## 技能内文档

- `docs/guides/`
- `docs/tutorial/`
- `docs/reference/`

## 说明

- 该技能是通用 Typst 能力；如果是演示文稿场景，应优先考虑配套的 `touying`。
