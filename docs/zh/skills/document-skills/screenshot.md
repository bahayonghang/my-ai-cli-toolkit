# 截图 (Screenshot)

跨平台截图技能，用于桌面级或系统级截图，适合在工具专属截图能力不可用或不合适时使用。

## 适用场景

- 用户明确要求截取桌面或系统截图
- 需要全屏、应用窗口、活动窗口或像素区域截图
- 浏览器或应用专属截图工具不可用、不足够，或不适合当前场景

## 工作流程

1. 根据用户要求或默认规则确定保存位置
2. 优先使用能直接访问目标的工具专属截图能力
3. 否则调用内置的跨平台截图脚本
4. 按要求截取全屏、应用窗口、活动窗口或局部区域
5. 返回保存路径，并在需要时查看截图结果

## 主要资源

- `scripts/take_screenshot.py`：macOS / Linux 截图脚本
- `scripts/take_screenshot.ps1`：Windows 截图脚本
- `scripts/ensure_macos_permissions.sh`：macOS 录屏权限预检查脚本

## 说明

- 在 macOS 上，该技能提供权限预检查流程，用于减少重复授权提示。
- 在 Linux 和 Windows 上，具体截图能力取决于系统可用工具。
- 如果用户提供保存路径，截图应保存到指定位置；否则使用系统默认截图位置，或在检查场景下保存到临时目录。
