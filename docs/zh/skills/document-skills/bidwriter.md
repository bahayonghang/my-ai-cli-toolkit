# BidWriter

面向工程咨询、建筑设计、市政工程等场景的标书编写技能。

## 适用场景

- 技术标或商务标撰写
- 招标文件分析
- 评分标准提取与响应策略设计
- 工程类投标方案写作

## 核心流程

1. 收集项目基本信息、招标文件、投标类型、公司资料和格式要求
2. 分析资格条件与评分项
3. 把招标结构映射为工作目录与章节树
4. 按招标文件章节逐段响应撰写
5. 在交付前完成合规、得分和一致性复核

## 关键规则

不得编造公司资质、业绩或证书。缺失信息必须明确标记为待补充。
