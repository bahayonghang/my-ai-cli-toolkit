## 技能分类

### 设计技能
用于创建视觉资产和界面的技能：
- **article-cover** - 生成文章 SVG 封面
- **drawio** - AI 驱动的 Draw.io 图表，支持实时浏览器预览
- **excalidraw** - 手绘风格图表
- **frontend-design** - 生产级 UI
- **gemini-image** - AI 图像生成
- **ui-ux-pro-max** - 高级 UI/UX 设计智能 (外部)

### 开发技能
用于代码分析和生成的技能：
- **codex** - Codex CLI 代码分析
- **paper-replication** - 论文复现为代码
- **review-code** - 多维度代码审查
- **rust-cli-tui-developer** - Rust CLI 和 TUI 开发专家
- **lib-slint-expert** - Slint GUI 开发专家
- **vue-best-practices** - Vue 3 最佳实践
- **uv-expert** - uv 包管理器专家
- **gh-bootstrap** - GitHub 仓库配置
- **agent-browser** - 浏览器自动化 (外部)

### 研究技能
用于收集和组织信息的技能：
- **research** - 网络研究与引用

### 文档技能
用于创建技术文档的技能：
- **tech-blog** - 技术博客
- **tech-design-doc** - 设计文档
- **spec-interview** - 规格访谈
- **mermaid-expert** - Mermaid.js 图表专家

### 学术写作技能
用于学术论文和学位论文写作的技能：
- **IEEE-writing-skills** - IEEE 论文翻译润色
- **latex-paper-en** - 英文论文 LaTeX 助手
- **latex-thesis-zh** - 中文论文 LaTeX 助手
- **typst-paper** - Typst 论文助手

### 技能开发
用于创建和管理技能的技能：
- **claude-expert-skill-creator** - 创建生产级技能
- **mcp-to-skill** - 转换 MCP 服务器
- **skill-manager** - GitHub 技能生命周期管理
- **skill-evolution-manager** - 技能迭代优化
- **github-to-skills** - GitHub 仓库转技能
- **skill-seekers** - 从文档生成技能

### 知识管理
用于管理和搜索知识的技能：
- **memory-system** - 本地记忆系统，支持 Markdown 文件的语义搜索

### 工具
实用的工具和程序：
- **yt-dlp** - 视频下载器
- **planning-with-files** - 基于文件的复杂任务规划
- **external-skills-tui** - 管理外部技能
