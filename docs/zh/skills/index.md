## 技能分类 (Skill Categories)

### 设计技能 (Design Skills)
用于创建视觉资产和界面的技能：
- **article-cover** - 生成文章 SVG 封面
- **drawio** - AI 驱动的 Draw.io 图表生成
- **excalidraw** - 手绘风格图表
- **frontend-design** - 生产级 UI
- **gemini-image** - AI 图像生成
- **ui-ux-pro-max** - 高级 UI/UX 设计智能 (外部)

### 开发技能 (Development Skills)
用于代码分析和生成的技能：
- **codex** - Codex CLI 代码分析
- **paper-replication** - 论文复现为代码
- **paper-check** - 代码审查助手
- **gemini** - Gemini 增强推理
- **explore** - 快速代码搜索
- **agent-browser** - 浏览器自动化 (外部)

### 研究技能 (Research Skills)
用于收集和组织信息的技能：
- **research** - 网络研究与引用
- **librarian** - 文档研究员
- **multimodal-looker** - 视觉分析师

### 文档技能 (Documentation Skills)
用于创建技术文档的技能：
- **tech-blog** - 技术博客
- **tech-design-doc** - 设计文档
- **spec-interview** - 规格访谈
- **document-writer** - 技术文档撰写

### 学术写作技能 (Academic Writing Skills)
用于学术论文和学位论文写作的技能：
- **IEEE-writing-skills** - IEEE 论文翻译润色
- **latex-paper-en** - 英文论文 LaTeX 助手
- **latex-thesis-zh** - 中文论文 LaTeX 助手
- **typst-paper** - Typst 论文助手
- **git-commit-cn** - 中文 git commit 消息

### 技能开发 (Skill Development)
用于创建和管理技能的技能：
- **claude-expert-skill-creator** - 创建生产级技能
- **mcp-to-skill** - 转换 MCP 服务器
- **skill-manager** - 搜索安装社区技能
- **skill-seekers** - 从文档生成技能

### 工具 (Utilities)
实用的工具和程序：
- **yt-dlp** - 视频下载器
- **external-skills-tui** - 管理外部技能

### OMO Agents (多智能体系统)
专业智能体协同工作的多智能体系统。
- **sisyphus** - 主编排器
- **oracle** - 专家架构师
- **omo-agents** - 系统概览

## 安装 (Installation)

安装所有技能：

::: code-group
```bash [Linux/macOS]
python3 install.py install-all
```
```powershell [Windows]
python install.py install-all
```
:::
