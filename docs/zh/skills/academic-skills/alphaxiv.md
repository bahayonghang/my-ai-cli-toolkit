# AlphaXiv Paper Lookup

在 alphaxiv.org 上查找任意 arxiv 论文，获取结构化的 AI 生成概览。

## 适用场景

- 用户提供 arxiv 链接或论文 ID
- 快速解释或总结 arxiv 论文
- 无需阅读完整 PDF 即可获取 AI 生成的论文概览

## 工作流程

1. 从提供的 URL 或输入中提取论文 ID
2. 通过 alphaxiv API 解析论文
3. 获取 AI 生成的概览
4. 可选：获取全文以进行深入分析

## 主要资源

- alphaxiv.org API 集成

## 说明

- 无需认证。
- 支持多种语言，包括英文、中文、日文等。
