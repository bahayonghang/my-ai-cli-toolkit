# Specialist 家族

`content/agents/specialist/` 当前包含的领域专家型 agent 主要有：

- `code-simplifier`
- `css`
- `deployment-specialist`
- `performance-optimizer`
- `python`
- `spec-analyst`
- `spec-architect`
- `spec-developer`
- `spec-planner`
- `spec-reviewer`
- `spec-task-reviewer`
- `spec-tester`
- `spec-validator`
- `typescript-expert`
- `ui-ux-designer`

## 这一家族适合什么

- 提供窄领域专业能力，而不是工作流编排
- 承担定向审查、实现建议或专家支持
- 处理 spec、样式、部署、语言相关的专门问题
