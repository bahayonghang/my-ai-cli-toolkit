- `action-planning-agent`
- `cli-discuss-agent`
- `cli-execution-agent`
- `cli-explore-agent`
- `cli-lite-planning-agent`
- `cli-planning-agent`

- `context-search-agent`
- `debug-explore-agent`
- `doc-generator`
- `issue-plan-agent`
- `issue-queue-agent`
- `memory-bridge`
- `test-context-search-agent`
- `test-fix-agent`
- `ui-design-agent`
- `universal-executor`

## 这一家族适合什么

- 把任务拆成可执行步骤
- 收集上下文与代码线索
- 协调代码、文档、测试类子任务
- 承担 issue / workflow / session 级别的自动化编排
