# Import Session Summary

Load and apply context from a previous session's summary file.

## Instructions

1. Read the `session_context.md` file (or specified summary file) from the current directory.
2. Parse and understand the following sections:
   - **Session Goal**: What was being worked on
   - **Accomplishments**: What was completed
   - **Key Decisions & Context**: Important architectural choices and discoveries
   - **Pending Items**: What needs to be done next
   - **Key Files**: Which files are relevant

3. Acknowledge the imported context and ask what to work on next.

## Action

Please read the session summary file and restore the context. Then ask me what I'd like to continue working on.
