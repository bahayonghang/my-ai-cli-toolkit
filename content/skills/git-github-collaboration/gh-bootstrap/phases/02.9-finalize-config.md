## Output

- **Format**: 完整配置 JSON（内存保持）
- **Location**: 传递给 Phase 3
- **Usage**: 作为后续阶段的输入

### Output Schema

```typescript
interface ConfigJSON {
  version: string;
  mode: 'quick' | 'custom' | 'full';
  project: {
    name: string;
    description: string;
    author: { name: string; email: string };
    license: string;
  };
  github: {
    owner: string;
    repo: string;
  };
  detected: {
    language: string;
    languages: string[];
    framework: string | null;
    frameworks: string[];
    packageManager: string;
    nodeVersion?: string;
    pythonVersion?: string;
    monorepo: string | null;
  };
  components: {
    infrastructure: string[];
    workflows: string[];
    collaboration: string[];
    security: string[];
    community: string[];
    automation: string[];
    quality: string[];
  };
  templateSources: Record<string, {
    repo: string;
    style?: string;
  }>;
  componentVariables: Record<string, any>;
  variables: Record<string, string>;
}
```
