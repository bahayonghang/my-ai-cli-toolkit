# 语言服务器参考

常用编程语言的 LSP 服务器信息及安装方法。

## Python

**服务器**: Pyright
**安装**: `pip install pyright` 或 `npm install -g pyright`
**官方插件**: `pyright-lsp` (可用)

## TypeScript/JavaScript

**服务器**: typescript-language-server
**安装**: `npm install -g typescript-language-server typescript`
**官方插件**: `typescript-lsp` (可用)

## Go

**服务器**: gopls
**安装**: `go install golang.org/x/tools/gopls@latest`
**官方插件**: 无 (需自定义)

## Rust

**服务器**: rust-analyzer
**安装**: https://rust-analyzer.github.io/manual.html#installation
**官方插件**: `rust-lsp` (可用)

## C/C++

**服务器**: clangd
**安装**: 
- Ubuntu/Debian: `apt install clangd`
- macOS: `brew install llvm`
- Windows: 下载 LLVM
**官方插件**: 无 (需自定义)

## Java

**服务器**: Eclipse JDT Language Server
**安装**: 下载并配置 Eclipse JDT LS
**官方插件**: 无 (需自定义)

## PHP

**服务器**: Intelephense
**安装**: `npm install -g intelephense`
**官方插件**: 无 (需自定义)

## Ruby

**服务器**: Solargraph
**安装**: `gem install solargraph`
**官方插件**: 无 (需自定义)

## CSS

**服务器**: vscode-css-languageserver
**安装**: `npm install -g vscode-langservers-extracted`
**官方插件**: 无 (需自定义)

## HTML

**服务器**: vscode-html-languageserver
**安装**: `npm install -g vscode-langservers-extracted`
**官方插件**: 无 (需自定义)
