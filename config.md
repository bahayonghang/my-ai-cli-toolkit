# 从零开始配置 Claude Code

## 1. 安装 Claude Code
使用如下命令安装配置基础配置：

```shell
npx zcf
```
使用官方安装和配置exa mcp

## 2. 配置 Claude 全局记忆

使用 prompts 中的 CLAUDE.md。

## 3. 安装 oh my claudecode

```shell
/plugin marketplace add https://github.com/Yeachan-Heo/oh-my-claudecode

/plugin install oh-my-claudecode

/oh-my-claudecode:omc-setup
```
