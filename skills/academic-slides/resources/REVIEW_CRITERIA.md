# Review Criteria

## Scoring Dimensions (5 × 10 = 50 pts)

### 1. Content Density (10 pts)
- Each slide ≤ 6 bullet points
- Each slide ≤ 80 words (EN) / 120 characters (CN)
- No overly crowded or empty slides
- Deductions: -1 per violation

### 2. Structure Completeness (10 pts)
Checklist (points per item):
- [2] Title page (title, author, institution, date)
- [1] Outline/TOC page
- [2] Introduction/background section
- [2] Core content (≥ 2 sections)
- [2] Summary/conclusion page
- [1] Q&A / acknowledgment page
- Bonus: references section (academic presentations)

### 3. Visual Balance (10 pts)
- Start at 10
- -2 if >3 consecutive text-only slides
- -1 per text-heavy slide (>5 bullets, no visuals)
- -1 if no tables or math formulas at all

### 4. Animation Usage (10 pts)
- Start at 10
- -2 if no animations at all
- -1 per slide with >3 animation commands (over-animation)
- -2 if <30% of content slides have animation

### 5. Compilation (10 pts)
- 10: compiles successfully with no errors
- 5: compiler not available (cannot verify)
- 0: compilation fails

## Grade Scale

| Grade | Score | Description |
|-------|-------|-------------|
| A | 46-50 | Excellent, ready to present |
| B | 36-45 | Good, minor improvements needed |
| C | 26-35 | Fair, significant improvements needed |
| D | ≤25 | Poor, consider regenerating |
