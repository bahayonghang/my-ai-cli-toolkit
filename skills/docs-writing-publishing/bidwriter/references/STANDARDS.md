## 引用规范

1. **格式** — 首次引用使用完整格式：`《标准名称》(标准编号)`，如 `《建筑设计防火规范》(GB 50016-2014)`
2. **版本** — 注意标注修订年版，如 `(2018年版)`
3. **有效性** — 标准可能被修订或废止，编写时应向用户确认现行版本
4. **地方标准** — 如需引用地方标准（DB/DBJ），应注明省份和适用范围
5. **团体标准** — T/ 开头的团体标准效力低于国标和行标，引用时应说明
6. **法律体系区分** — 工程建设项目适用《招标投标法》及其实施条例；政府采购货物与服务适用《政府采购法》及财政部令第 87 号等，引用时勿混用
