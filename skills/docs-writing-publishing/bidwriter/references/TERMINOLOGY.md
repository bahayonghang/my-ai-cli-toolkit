# 领域术语词典

标书编写中常用的专业术语速查表，按领域分类。编写时应优先使用本表中的规范表述。

## 建筑设计术语

| 术语           | 英文                          | 释义                                       |
| -------------- | ----------------------------- | ------------------------------------------ |
| 容积率         | Floor Area Ratio (FAR)        | 地上总建筑面积与用地面积之比               |
| 建筑密度       | Building Coverage Ratio       | 建筑基底面积与用地面积之比                 |
| 绿地率         | Green Space Ratio             | 绿地面积与用地面积之比                     |
| 建筑高度       | Building Height               | 室外地坪至建筑最高点的垂直距离             |
| 日照间距       | Sunshine Spacing              | 保证后排建筑底层获得规定日照时间的最小间距 |
| 消防登高场地   | Fire Truck Access Area        | 供消防车登高作业的场地                     |
| 防火分区       | Fire Compartment              | 在建筑内部采用防火墙、楼板等划分的区域     |
| 疏散宽度       | Evacuation Width              | 安全出口和疏散通道的最小净宽度             |
| 无障碍设计     | Barrier-free Design           | 方便残障人士使用的建筑设计                 |
| 装配式建筑     | Prefabricated Building        | 采用预制构件在工地装配而成的建筑           |
| 绿色建筑       | Green Building                | 在全生命周期内节约资源、保护环境的建筑     |
| BIM 技术       | Building Information Modeling | 建筑信息模型技术                           |
| 被动式超低能耗 | Passive Ultra-low Energy      | 通过建筑本体性能提升降低能耗的建筑技术     |
| 海绵城市理念   | Sponge City Concept           | 城市雨水自然积存、渗透、净化的建设理念     |
| 竖向设计       | Vertical Design               | 场地或道路纵向高程的规划设计               |

## 市政工程术语

| 术语         | 英文                            | 释义                                   |
| ------------ | ------------------------------- | -------------------------------------- |
| 路基         | Subgrade / Roadbed              | 道路路面以下承受荷载的土体结构         |
| 路面结构层   | Pavement Structure              | 面层、基层、垫层组成的道路结构         |
| 综合管廊     | Utility Tunnel                  | 集中敷设多种市政管线的地下空间         |
| 雨污分流     | Separate Sewer System           | 雨水和污水分别由不同管道排放           |
| 合流制       | Combined Sewer System           | 雨水和污水由同一管道排放               |
| 顶管施工     | Pipe Jacking                    | 非开挖条件下敷设地下管道的施工方法     |
| 基坑支护     | Foundation Pit Support          | 保证基坑开挖稳定性的临时支撑结构       |
| 沥青混凝土   | Asphalt Concrete (AC)           | 由沥青、粗细集料和矿粉组成的路面材料   |
| 水泥稳定碎石 | Cement Stabilized Macadam       | 水泥与碎石混合压实的路面基层材料       |
| 交通渠化     | Traffic Channelization          | 通过标线和设施引导车流的交通组织方式   |
| 管网水力计算 | Pipe Network Hydraulic Analysis | 管道系统流量、流速、水压的计算分析     |
| 初期雨水     | First Flush                     | 降雨初期污染浓度较高的径流             |
| LID 设施     | Low Impact Development          | 低影响开发设施（透水铺装、雨水花园等） |
| CCTV 检测    | CCTV Inspection                 | 管道闭路电视检测技术                   |
| 道路红线     | Road Red Line                   | 规划道路用地的边界线                   |

## 工程造价术语

| 术语       | 英文                     | 释义                                     |
| ---------- | ------------------------ | ---------------------------------------- |
| 工程量清单 | Bill of Quantities (BOQ) | 按统一规则计算的分部分项工程量汇总表     |
| 综合单价   | Comprehensive Unit Price | 包含人工、材料、机械、管理费和利润的单价 |
| 暂列金额   | Provisional Sum          | 招标人预留的用于施工中可能变更的金额     |
| 暂估价     | Provisional Price        | 招标人暂定的材料或专业工程单价           |
| 措施项目费 | Temporary Works Cost     | 施工过程中的安全文明施工等措施费用       |
| 规费       | Regulatory Fee           | 按规定应缴纳的社会保险和住房公积金       |
| 投标限价   | Bid Ceiling Price        | 招标人设定的最高投标报价                 |
| 投标报价   | Bid Price                | 投标人根据工程量清单提出的总价           |
| 清单计价   | BOQ Pricing              | 按工程量清单计价规范进行的工程计价方式   |
| 定额计价   | Quota Pricing            | 按预算定额进行的传统工程计价方式         |
| 价差调整   | Price Adjustment         | 因材料价格波动对合同价格进行的调整       |
| 竣工结算   | Final Settlement         | 工程完工后对实际费用的最终核算           |

## 招投标术语

| 术语                 | 英文                     | 释义                                   |
| -------------------- | ------------------------ | -------------------------------------- |
| 招标人               | Tenderer / Employer      | 发出招标邀请的单位（通常为业主方）     |
| 投标人               | Bidder                   | 响应招标、提交投标文件的单位           |
| 招标代理机构         | Tendering Agency         | 受招标人委托组织招标活动的中介机构     |
| 评标委员会           | Bid Evaluation Committee | 负责评审投标文件的专家委员会           |
| 投标保证金           | Bid Bond / Bid Security  | 投标时缴纳的保证参与投标诚信的担保金   |
| 履约保证金           | Performance Bond         | 中标后缴纳的保证合同履行的担保金       |
| 综合评估法           | Comprehensive Evaluation | 对技术、商务、价格综合打分的评标方法   |
| 经评审的最低投标价法 | Lowest Evaluated Bid     | 在满足要求前提下选择最低报价的评标方法 |
| 资格预审             | Prequalification         | 在投标前对投标人资格条件的审查         |
| 资格后审             | Post-qualification       | 在开标后对投标人资格条件的审查         |
| 废标                 | Bid Rejection            | 不符合招标文件要求被否决的投标         |
| 澄清                 | Clarification            | 对投标文件中不明确内容的书面说明       |
| 中标通知书           | Letter of Acceptance     | 通知中标人中标结果的书面文件           |
| 合同谈判             | Contract Negotiation     | 中标后就合同细节进行的协商             |
| 联合体投标           | Joint Venture Bid        | 两个以上单位组成联合体共同投标         |
| 分包                 | Subcontracting           | 将部分工作委托给其他单位完成           |
| 响应性               | Responsiveness           | 投标文件对招标要求的符合程度           |
| 偏离表               | Deviation Schedule       | 列出投标方案与招标要求差异的表格       |

| 微服务       | Microservices                 | 将系统拆分为可独立部署的小型服务         |
| 容灾备份     | Disaster Recovery & Backup    | 数据与系统的灾难恢复与备份机制           |

| 系统集成     | System Integration            | 将不同子系统/设备整合为统一系统          |

## 政府采购术语

| 术语         | 英文                            | 释义                                       |
| ------------ | ------------------------------- | ------------------------------------------ |
| 政府采购     | Government Procurement          | 国家机关、事业单位采购货物/工程/服务的行为 |
| 公开招标     | Open Tendering                  | 政府采购的主要方式，公开邀请不特定供应商   |
| 竞争性谈判   | Competitive Negotiation         | 与多家供应商谈判后确定成交的采购方式       |
| 竞争性磋商   | Competitive Consultation        | 经磋商后报价、综合评分的政府采购方式       |
| 单一来源采购 | Single-source Procurement       | 只能从唯一供应商处采购的方式               |
| 询价         | Request for Quotation           | 对规格统一的现货货物的简易采购方式         |
| 框架协议采购 | Framework Agreement             | 先确定入围供应商再分批下单的采购方式       |
| 供应商       | Supplier / Vendor               | 政府采购中提供货物或服务的投标人           |
| 采购人       | Procuring Entity                | 实施政府采购的国家机关、事业单位等         |
| 集中采购目录 | Centralized Procurement Catalog | 必须委托集中采购机构采购的品目清单         |

## 使用规范

1. **首次出现** 时使用全称，后续可使用简称或缩写
2. **同一文档** 中术语表述必须前后一致
3. **引用标准** 时使用标准的正式名称和编号
4. **地方术语** 与国标术语不同时，优先使用国标术语并注明地方叫法
5. **新兴术语**（如 BIM、CIM、数字孪生）需简要释义后使用
